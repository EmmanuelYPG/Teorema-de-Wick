# Teorema-de-Wick
- Código que genera las combinaciones necesarias del Teorema de Wick, fundamental en teoría cuántica de campos QFT, en el cual puedes seleccionar el número de puntos y el orden de lazo deseado.

- Code that generates the necessary combinations of Wick's Theorem, which is fundamental to quantum field theory QFT, allowing you to select the number of vertices and the desired loop order.
